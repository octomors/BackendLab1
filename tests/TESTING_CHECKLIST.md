# Чек-лист тестирования RecipeService

## 1. Метод `get_paginated_recipes`

### Базовые сценарии
- [ ] Получение списка рецептов без фильтров
- [ ] Пагинация работает корректно

### Фильтрация по имени (name__like)
- [ ] Поиск рецептов по частичному совпадению названия
- [ ] Поиск без результатов возвращает пустой список
- [ ] Регистронезависимый поиск

### Фильтрация по ингредиентам (ingredient_id)
- [ ] Фильтрация по одному ингредиенту
- [ ] Фильтрация по нескольким ингредиентам
- [ ] Фильтрация по несуществующему ингредиенту возвращает пустой результат

### Сортировка (sort)
- [ ] Сортировка по возрастанию (по id)
- [ ] Сортировка по убыванию (-id)
- [ ] Сортировка по другим полям (title, cooking_time)

## 2. Метод `get_recipes_by_ingredient`

### Базовые сценарии
- [ ] Получение рецептов с указанным ингредиентом
- [ ] Возврат пустого списка, если рецептов с ингредиентом нет

### Обработка ошибок
- [ ] ValueError при несуществующем ингредиенте

### Параметр include
- [ ] Включение cuisine в ответ
- [ ] Включение author в ответ
- [ ] Включение allergens в ответ
- [ ] Включение ingredients в ответ
- [ ] Включение нескольких связанных сущностей

### Параметр select_fields
- [ ] Выбор конкретных полей из ответа
- [ ] Комбинация include и select_fields

## 3. Метод `build_recipe_response`

### Базовые сценарии
- [ ] Формирование полного ответа для рецепта
- [ ] Корректное включение информации о кухне (cuisine)
- [ ] Корректное включение информации об авторе (author)
- [ ] Корректное включение списка аллергенов
- [ ] Корректное включение списка ингредиентов с количеством

### Граничные случаи
- [ ] Рецепт без cuisine (cuisine_id = None)
- [ ] Рецепт без аллергенов
- [ ] Рецепт без ингредиентов

## 4. Метод `build_recipe_response_selective`

### Базовые сценарии
- [ ] Формирование ответа с базовыми полями (без includes)
- [ ] Включение только cuisine
- [ ] Включение только author
- [ ] Включение только allergens
- [ ] Включение только ingredients
- [ ] Комбинация нескольких includes

### Граничные случаи
- [ ] Пустой список includes
- [ ] Включение несуществующего поля в includes (игнорируется)
