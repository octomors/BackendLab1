# Чек-лист тестирования RecipeService

## 1. Метод `get_paginated_recipes`

### Базовые сценарии
- [x] Получение списка рецептов без фильтров
- [x] Пагинация работает корректно

### Фильтрация по имени (name__like)
- [x] Поиск рецептов по частичному совпадению названия
- [x] Поиск без результатов возвращает пустой список
- [x] Регистронезависимый поиск

### Фильтрация по ингредиентам (ingredient_id)
- [x] Фильтрация по одному ингредиенту
- [x] Фильтрация по нескольким ингредиентам
- [x] Фильтрация по несуществующему ингредиенту возвращает пустой результат

### Сортировка (sort)
- [x] Сортировка по возрастанию (по id)
- [x] Сортировка по убыванию (-id)
- [x] Сортировка по другим полям (title, cooking_time)

### Комбинированные фильтры
- [x] Фильтрация по имени и ингредиенту одновременно

## 2. Метод `get_recipes_by_ingredient`

### Базовые сценарии
- [x] Получение рецептов с указанным ингредиентом
- [x] Возврат пустого списка, если рецептов с ингредиентом нет

### Обработка ошибок
- [x] ValueError при несуществующем ингредиенте

### Параметр include
- [x] Включение cuisine в ответ
- [x] Включение author в ответ
- [x] Включение allergens в ответ
- [x] Включение ingredients в ответ
- [x] Включение нескольких связанных сущностей

### Параметр select_fields
- [x] Выбор конкретных полей из ответа
- [x] Комбинация include и select_fields

## 3. Метод `build_recipe_response`

### Базовые сценарии
- [x] Формирование полного ответа для рецепта
- [x] Корректное включение информации о кухне (cuisine)
- [x] Корректное включение информации об авторе (author)
- [x] Корректное включение списка аллергенов
- [x] Корректное включение списка ингредиентов с количеством

### Граничные случаи
- [x] Рецепт без cuisine (cuisine_id = None)
- [x] Рецепт без аллергенов
- [x] Рецепт без ингредиентов

### Базовые сценарии
- [x] Формирование ответа с базовыми полями (без includes)
- [x] Включение только cuisine
- [x] Включение только author
- [x] Включение только allergens
- [x] Включение только ingredients
